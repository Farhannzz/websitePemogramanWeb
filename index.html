Module 01

Sistem Operasi: Fundamental

Sistem Operasi (Operating System) adalah perangkat lunak yang berfungsi sebagai pengelola sumber daya komputer dan sebagai penghubung antara pengguna dengan perangkat keras komputer.

Fungsi Utama Sistem Operasi

  • Manajemen proses
  • Manajemen memori
  • Manajemen file dan penyimpanan
  • Manajemen input/output
  • Pengaturan keamanan dan proteksi sistem

Sistem operasi memastikan komputer dapat digunakan dengan nyaman, stabil, dan efisien.

Module 02

Jenis & Fungsi Sistem Operasi

Sistem operasi memiliki berbagai jenis sesuai kebutuhan perangkat. Pada dasarnya, sistem operasi bertugas mengatur jalannya program dan mengelola sumber daya komputer.

Jenis Sistem Operasi

Desktop OS

Windows, Linux, macOS

Mobile OS

Android, iOS

Server OS

Ubuntu Server, Windows Server

Embedded OS

OS pada IoT, mesin ATM, Smart TV

Fungsi Utama

  • Menyediakan antarmuka pengguna
  • Mengatur eksekusi program
  • Menangani perangkat keras
  • Menjaga keamanan sistem
Module 03

Proses dan Thread

Proses adalah program yang sedang dieksekusi dan memiliki ruang memori tersendiri. Sedangkan thread adalah unit terkecil dari proses yang dapat dijalankan secara independen.

Perbedaan Proses & Thread

FiturProsesThread
ResourceButuh resource lebih besarRingan & cepat
MemoriMemiliki ruang memori sendiriBerbagi memori dengan proses utama

Manfaat Threading

  • Meningkatkan kinerja multitasking
  • Mempercepat eksekusi aplikasi
  • Cocok untuk aplikasi paralel
Module 04

Manajemen Proses

Sistem operasi bertanggung jawab mengatur proses mulai dari dibuat, dijalankan, dihentikan hingga dihapus dari memori.

Status Proses

New → Ready → Running → Waiting → Terminated

PCB (Process Control Block): Struktur data yang menyimpan informasi penting proses seperti state, program counter, register dan memori.

Module 05

Penjadwalan CPU

Penjadwalan CPU menentukan proses mana yang akan mendapat giliran menggunakan CPU. Tujuannya untuk efisiensi dan mempercepat waktu respon.

Algoritma Penjadwalan CPU

  • FCFS (First Come First Serve)
  • SJF (Shortest Job First)
  • Round Robin
  • Priority Scheduling

Pemilihan algoritma sangat mempengaruhi performa sistem operasi.

Module 06

Sinkronisasi Proses

Sinkronisasi proses diperlukan ketika beberapa proses mengakses data yang sama agar tidak terjadi konflik.

Konsep Penting

  • Race Condition: dua proses berebut resource
  • Critical Section: bagian kode yang tidak boleh dipakai bersamaan
  • Semaphore & Mutex: alat pengendali akses

Tanpa sinkronisasi, data mudah rusak/invalid.

Module 07

Deadlock

Deadlock terjadi ketika dua atau lebih proses saling menunggu resource satu sama lain dan tidak ada yang dapat melanjutkan eksekusi.

Syarat Deadlock

Mutual Exclusion, Hold and Wait, No Preemption, Circular Wait

Pencegahan & Penanganan

  • Resource Allocation Graph
  • Banker’s Algorithm
  • Deteksi & Recovery
Module 08

Memori Utama

Memori utama digunakan untuk menyimpan data dan instruksi yang sedang diakses CPU.

Kecepatan memori sangat berpengaruh pada performa sistem.

Jenis Memori

  • RAM (Random Access Memory)
  • ROM (Read Only Memory)
Module 09

Manajemen Memori

Manajemen memori mengatur pembagian ruang memori agar setiap proses dapat berjalan secara efisien tanpa saling mengganggu.

Teknik Manajemen

  • Paging: membagi memori menjadi frame kecil
  • Segmentation: pembagian berdasarkan struktur program
  • Swapping: memindahkan proses dari RAM ke disk sementara
Module 10

Virtual Memory

Virtual Memory memungkinkan sistem menjalankan program yang ukurannya lebih besar daripada kapasitas RAM dengan memanfaatkan media penyimpanan sekunder.

Komponen Utama

  • Paging
  • Page Table
  • Page Fault
  • Demand Paging
Module 11

Sistem Berkas (File System)

Sistem berkas bertugas mengelola penyimpanan, pengorganisasian, serta pengaksesan file pada perangkat komputer.

Operasi File

Create, Read, Write, Delete, Rename

Struktur Direktori

Single Level, Two Level, Tree Directory, Graph Directory

Manajemen I/O

Tujuan Manajemen I/O

  • Menangani interrupt dari perangkat

Module 18

Shell dan Command Line

Shell adalah antarmuka teks untuk berinteraksi dengan sistem operasi melalui perintah.

Perintah Dasar CLI

  • cd → pindah direktori
  • ls/dir → melihat file
  • cp → copy file
  • rm → hapus file
